# Ubuntu16.04 Atom 使用记录
***
## 一、安装 Atom
```
sudo add-apt-repository ppa:webupd8team/atom
sudo apt-get update
sudo apt-get install atom
```

## 二、卸载 Atom
```
sudo apt-get remove atom
sudo apt-get autoremove
sudo add-apt-repository --remove ppa:webupd8team/atom
```
